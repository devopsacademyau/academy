- Answer the following questions
  - What is the difference between inbound x outbound?
    ```
    inbound is traffic entering your network and outbound is traffic exiting the network. 
    ```
    
  - What is going to happen by default? Everything allowed or denied?
    ```
    By default all traffic in and out is allowed but you can adjust the number on the side (lower is higher priority) if you want to adjust that. 
    ```
